# Product Review Scrapper

Simple review scrapper for scrapping products review from flipkart website

## Technologies

  - Python
  - Flask
  - Beautiful Soup

 - [aws](http://reviewscrapper-env.eba-srmtmk53.us-east-2.elasticbeanstalk.com/)
